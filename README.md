# Ginge Dancewear Marketplace

A high-end, editorial marketplace concept for curated dancewear resale.

## Included pages

- `/` — Luxury Ginge homepage
- `/shop` — Searchable inventory catalog with collection tabs, filters, saved items, and concept search
- `/sell` — Five-step Seller Studio demonstrating photo preparation, AI listing creation, price guidance, and listing review

## Run locally

Requirements: Node.js 22.13 or newer and npm.

```bash
npm install
npm run dev
```

Open the local address shown in the terminal.

## Production build

```bash
npm run build
```

## Upload to GitHub

1. Create a new empty repository in GitHub.
2. Extract this package.
3. Upload the extracted files to the repository, or push them with Git.
4. Do not upload `node_modules`, `dist`, or `.sites-runtime` if you create them locally.

## Current scope

This package is a polished interactive front-end concept. The sample inventory and seller flow use local demonstration data. Production marketplace development would connect user accounts, permanent image storage, AI services, payments, shipping, moderation, and a database.

## Technology

- React 19
- Next.js-compatible app structure
- Vinext/Vite production runtime
- TypeScript
- Responsive CSS

All branded imagery included in this package was created specifically for the Ginge concept.
